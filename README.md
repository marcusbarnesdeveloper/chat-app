# chat-app

An application that allows user to create, join and have conversations in chat rooms
![demo](demo/demo.gif)

## Technology used

1. Javascript
2. NodeJs
3. Express
4. Socket.io
5. HTML5
6. CSS3

## Requirements

- Node 12.0.0

### Installing Dependencies

From within the root directory:

```sh
npm install
```

To serve the module on localhost:3002:

```sh
npm start
```
